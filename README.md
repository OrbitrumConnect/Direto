## 🚀 OPÇÕES DE DEPLOY

### 🔄 **HÍBRIDO** (Recomendado):
```bash
cd frontend && vercel --prod     # Frontend no Vercel
cd backend && railway up         # Backend no Railway
```

### 🟢 **VERCEL ONLY** (Fullstack):
```bash
vercel --prod                    # Tudo no Vercel
```

### 🔵 **RAILWAY ONLY** (Fullstack):
```bash
railway up                       # Tudo no Railway
```
